# Local Storage

Local Storage creates helpers to work with specific items inside local storage

## Installation

```
npm i @rocketstation/local-storage
```

## Usage

```javascript
import LS from '@rocketstation/local-storage'

const ls = new LS('key)

ls.get()
ls.remove()
ls.set({})
```
